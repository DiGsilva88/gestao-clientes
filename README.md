# 📋 Gestão de Clientes e Fornecedores

Aplicação web desenvolvida em Laravel para gestão de clientes e fornecedores.

## 🚀 Tecnologias utilizadas

- PHP 8.2
- Laravel 12
- MySQL
- Bootstrap 5
- Bootstrap Icons

## ⚙️ Instalação

1. Clona o repositório:
```bash
git clone https://github.com/teu-utilizador/nome-do-repo.git
```

2. Instala as dependências:
```bash
composer install
npm install
```

3. Copia o ficheiro de ambiente:
```bash
cp .env.example .env
```

4. Gera a chave da aplicação:
```bash
php artisan key:generate
```

5. Configura a base de dados no `.env`:

6. Corre as migrations:
```bash
php artisan migrate
```

7. Inicia o servidor:
```bash
php artisan serve
```

## 📌 Funcionalidades

- ✅ CRUD de Clientes
- ✅ CRUD de Fornecedores
- ✅ Pesquisa por nome e email
- ✅ Autenticação (login/registo)
- ✅ Navbar responsiva
